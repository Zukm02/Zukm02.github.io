Drug Patterns By State Capstone Project

This project evaluates how state-level substance use laws affect public health outcomes using real-world admission and overdose data. Using a difference-in-difference regression framework, I compared trends across states with and without involuntary commitment laws.

🧪 Methods Used

  • Data cleaning and aggregation with Python (Pandas)
  • Difference-in-difference regression modeling
  • Visualization using Matplotlib and Seaborn

📊 Key Visuals

📌 Takeaways

States with involuntary commitment laws showed statistically significant differences in treatment admissions over time. Results suggest policy may be a key factor in public health trends and resource demand.
